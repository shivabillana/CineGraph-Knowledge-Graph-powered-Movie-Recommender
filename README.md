# 🎬 CineGraph — Knowledge Graph Movie Recommender

A movie recommendation system built on a **Neo4j Knowledge Graph**, with a **FastAPI** backend and **Streamlit** frontend. Instead of collaborative filtering or embeddings, CineGraph uses graph traversal to find similar movies based on shared genres, directors, and production companies.

---

## 🧠 How It Works

```
CSV Dataset → Neo4j AuraDB (Knowledge Graph)
                      ↓
             FastAPI Recommendation API
                      ↓
           Streamlit Frontend (User Interface)
```

Movies are connected to Directors, Genres, and Companies as graph nodes. Recommendations are generated by traversing these relationships and scoring movies based on shared connections.

**Scoring Logic:**
| Signal | Weight |
|---|---|
| Shared Genre | +1 per genre |
| Shared Director | +2 (stronger signal) |
| Shared Company | +1 per company |

## ⚙️ Setup

### 1. Clone the repo
```bash
git clone https://github.com/yourusername/cinegraph.git
cd cinegraph
```

### 2. Create and activate virtual environment
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure environment variables
Create a `.env` file in the root directory:
```env
NEO4J_URI=neo4j+s://your-instance-id.databases.neo4j.io
NEO4J_USERNAME=neo4j
NEO4J_PASSWORD=your-aura-password
```

### 5. Run data insertion (first time only)
```bash
python data_insertion.py
```

## 🚀 Running the App

You need **two terminals** running simultaneously:

**Terminal 1 — Start FastAPI backend:**
```bash
uvicorn main:app --reload
```
API runs at: `http://localhost:8000`
Swagger docs: `http://localhost:8000/docs`

**Terminal 2 — Start Streamlit frontend:**
```bash
streamlit run app.py
```
UI runs at: `http://localhost:8501`

## 🔌 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/` | Health check |
| GET | `/movies/search?q=inception` | Search movies by name |
| GET | `/recommend?movie=Inception` | Combined recommendations |
| GET | `/recommend/genre?movie=Inception` | Genre-based recommendations |
| GET | `/recommend/director?movie=Inception` | Director-based recommendations |
| GET | `/recommend/company?movie=Inception` | Company-based recommendations |

All recommendation endpoints accept an optional `limit` parameter (default: 10).

## 🗃️ Knowledge Graph Schema

```
(Movie)-[:DIRECTED_BY]->(Director)
(Movie)-[:HAS_GENRE]->(Genre)
(Movie)-[:PRODUCED_BY]->(Company)
```
